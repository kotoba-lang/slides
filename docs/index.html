kotoba-lang/slides

Portable CLJC model for slides, docs, drive, and sheets. The runtime surface is pure EDN, and the JVM host can write a minimal PowerPoint package directly from the same deck data.

EDN to PPTX

Use slides.model/deck, slides.model/slide, and shape maps, then call slides.pptx/write-pptx!. The writer emits OPC relationships, presentation parts, slide XML, theme, master, and layout XML without a JavaScript PPTX runtime.
